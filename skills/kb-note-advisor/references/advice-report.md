# Advisory Report Template

Use this template when the user asks to save a durable review.

```markdown
---
type: advice
reviewed_note:
created: YYYY-MM-DD
status: open
tags: []
---

# Advisory Review: Note Title

Reviewed note: [[personal-notes/path/to-note]]

## Scope

## Likely Correct

## Possibly Incomplete

## Possibly Wrong or Contradictory

## Missing Evidence or Sources

## Suggested Revisions

## Questions Worth Investigating

## Sources Consulted
```

## Finding Format

```markdown
### Short finding title

- Note claim:
- Assessment:
- Evidence:
- Suggested revision:
- Confidence:
```

## Review Heuristics

- A note is incomplete when it omits a major mechanism, limitation, alternative explanation, or boundary condition present in the sources.
- A note is possibly wrong when it contradicts a raw-source record or wiki synthesis with stronger evidence.
- A note is under-evidenced when it makes a broad claim from personal experience or a single source without enough support.
- A note may still be valuable when it records personal context, intuition, or local experience; clarify scope instead of dismissing it.
- Prefer exact replacement wording over vague advice such as "add more detail."
